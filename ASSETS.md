# Assets notes

These images are production-quality **raster concept assets** to accelerate implementation and visual alignment.

Important:
- Use them directly for the first implementation if needed.
- For the final polished brand system, the **logo and monogram should ideally be redrawn as SVG** by a designer or an implementation agent working from these references.
- Keep the overall composition, palette, and visual hierarchy, but prefer vector output for:
  - favicon / app icon
  - navbar logo
  - footer logo
  - slide and document headers
  - high-DPI / Retina rendering

Recommended priority order:
1. `assets/logos/arkham-primary-dark.png`
2. `assets/logos/arkham-primary-light.png`
3. `assets/logos/arkham-horizontal-dark.png`
4. `assets/logos/arkham-monogram-dark.png`
5. `assets/logos/arkham-monogram-light.png`
6. `assets/social/arkham-og-1200x630.png`
7. `assets/hero/arkham-hero-1920x1080.webp`

If the implementation agent redraws the identity:
- preserve the serif `A`
- preserve the single tentacle sweep
- preserve the subtle astrolabe / compass geometry
- do not add extra horror details
- keep the mood editorial and premium, not theatrical
